"Responsive" SVG

SVG supports <style>. Meaning, we can use svg with <img> and an embedded >style< tag to create a "responsive" graphic.

In this case, we are creating a semi dynamic, but non interactive svg file. Also, the viewport in these embedded tags is not the browser window, but the SVG canvas size, which, in this instance, is beneficial.

These styles live in their own world and do not interact with the user/browser. ie. - :hover will not work.
